# Ineco Performance Marketing Dashboard Design v2.0

## Overview

**Purpose:** Enable data-driven marketing decisions for Ineco Bank  
**Users:** CMO, Marketing Manager, Performance Marketers, Product Managers  
**Data Sources:** GA4 (website), Ad Platforms (spend), Bank Systems (approvals + revenue)

**Design Principles:**
- 25 charts maximum (focused, not overwhelming)
- All trends show explicit time comparison (WoW, MoM, YoY)
- Target/budget variance on all KPIs
- Revenue focus (not just registrations)
- Alert thresholds for decision-readiness
- Cohort view for approval lag analysis

```sql
CREATE TABLE `ineco_config.targets` (
  period DATE,           -- First day of month
  metric STRING,         -- 'revenue', 'approved', 'cpa', 'roas', 'approval_rate'
  target_value FLOAT64,
  channel STRING         -- 'all', 'google_ads', 'meta_ads', etc.
);
```

```sql
CREATE TABLE `ineco_config.budgets` (
  period DATE,           -- First day of month

  monthly_budget FLOAT64
);
```
